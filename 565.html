Errata for Practical Python

Chapter 1

On page 26,

Stringa are values [...]

should be

Strings are values [...]

Chapter 2

On page 111,

[...] just change while not name to while name.isspace().

should be

[...] just change while not name to while not name or name.isspace().

or, alternatively,

[...] just change while not name to while not name.strip().

Chapter 3

On page 64,

>>> print format % value

should be

>>> print format % values

Appendix B

On page 580, in Table B-6, the description of the method startswith uses the word suffix where it should use the word prefix. The description should read:

Checks whether string starts with prefix, optionally restricting the matching with the given indices start and end
